# Project Context: Hexo Theme "Material 3 Pure"

## 1. Role & Objective
**Role:** You are a Senior Frontend Developer and a Google Material Design 3 (Material You) Expert.
**Objective:** Build a Hexo theme from scratch that strictly follows the Material Design 3 guidelines.
**Visual Reference:** The theme must visually replicate the layout, typography, and spacing of the official [Material Design Blog](https://m3.material.io/blog).

## 2. Design Specifications (The "Source of Truth")

### 2.1 Layout Architecture
* **Desktop (Width > 900px):**
    * **Left:** Fixed Navigation Rail (Width: 80px collapsed / 360px expanded).
    * **Right:** Fluid Content Area (CSS Grid).
* **Mobile (Width < 900px):**
    * **Top:** Simple Top App Bar.
    * **Bottom:** Navigation Bar (Fixed bottom).
    * **Content:** Single column fluid.

### 2.2 Visual Language (Material You)
* **Color System:** strictly use Dynamic Color tokens via CSS Variables.
    * Background: `var(--md-sys-color-surface)`
    * Containers: `var(--md-sys-color-surface-container)`
    * Text: `var(--md-sys-color-on-surface)`
    * Accent: `var(--md-sys-color-primary)`
* **Shape & Radius:**
    * Cards: `border-radius: 24px;` (Large)
    * Buttons: `border-radius: 20px;` (Pill shape)
    * Images: `border-radius: 16px` to `24px`.
* **Elevation:** Avoid harsh shadows (box-shadow). Use **Surface Tones** (lighter/darker backgrounds) to indicate hierarchy.
* **Typography:** Use `Roboto` or `Open Sans`.
    * Headlines: High contrast, geometric.
    * Body: Readable, ample line-height.

### 2.3 Components Strategy
* **Web Components:** Prioritize using Google's official `@material/web` via CDN/ES Modules.
    * Target components: `<md-elevated-card>`, `<md-filled-button>`, `<md-icon>`, `<md-list>`.
* **CSS Grid:** Use Modern CSS Grid for the masonry article layout, mimicking the m3.material.io blog grid.

## 3. Technical Stack & Environment
* **Core:** Hexo (Node.js Static Site Generator).
* **Template Engine:** EJS (`.ejs`).
* **Styling:** Vanilla CSS with CSS Custom Properties (Variables).
* **Scripting:** Vanilla JavaScript (ES6 Modules).
* **External Assets:**
    * Google Fonts (Roboto).
    * Material Symbols Outlined (Icons).
    * Material Web (ESM CDN).
